Detailed Description

template<typename Factory, typename ConcreteProduct>
class generic_factory::Proxy< Factory, ConcreteProduct >

A proxy for registering into a factory. It provides the builder as static method and the automatic registration mechanism.

Template Parameters
Factorythe type of the factory
ConcreteProductit is the derived (concrete) type to be registered in the factory
Note
the default builder provided by the factory has to be used. However, no check it is made to verify it
