Parameters
etafunctional sparse matrix containing the basis of the functional regression coefficients of the non-stationary covariates (qncxLnc, row i-th contains zeros and the basis of the i-th non-stationary covariate, their position shifted of sum_i_0_to_i(Lnc_i))
qncnumber of non-stationary covariates
Lnctotal number of basis used for the non-stationary covariates functional regression coefficients
Lnc_jvector containing in element i-th the number of basis for the non-stationary covariate i-th functional regression coefficients
y_trainfunctional matrix containing the response of the training set (n_train x 1)
Xc_trainfunctional matrix containing the stationary covariates of the training set (n_train x qc)
Xnc_trainfunctional matrix containing the non-stationary covariates of the training set (n_train x qnc)
Rncpenalization matrix of the non-stationary covariates (diagonal block matrix containing the the scalar product within the second order derivatives of the functional regression coefficients basis. LncxLnc)
aleft extreme functional data domain
bright extreme functional data domain
n_intervals_integrationnumber of intervals used by the midpoint quadrature rule
n_trainnumber of training statistical units
number_threadsnumber of threads for OMP
