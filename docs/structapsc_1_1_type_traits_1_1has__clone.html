Detailed Description

template<typename T, typename Sfinae = void>
struct apsc::TypeTraits::has_clone< T, Sfinae >

A type trait than checks if your class contains a method called clone() the template parameter. I use it to implement clonable classes which enable the prototype design pattern on a polymorphic family of classes. This is the primary template, which maps to false.

Specialised version that is activated if T is clonable. Indeed, if T is not clonable the second template parameter cannot be substituted with a valid type. So SFINAE applies and this version is discarded. declval<T&>() allows to test the return type of clone() with no need of creating an object of type T.
