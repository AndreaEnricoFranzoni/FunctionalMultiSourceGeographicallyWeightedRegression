Detailed Description

template<typename T>
struct RowView< T >

Row-view: accessing to a row of a dense matrix of callable object with a pair of iterators.

Template Parameters
Tthe object type stored in the matrix
Note
since the matrices store object column-wise, the iterator with offset has to be used
